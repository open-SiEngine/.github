# Solid 系列车用操作系统内核与虚拟化平台

**Solid** 是一套面向智能汽车的开源、安全、实时操作系统内核及虚拟化解决方案，基于成熟的 **L4Re 微内核架构** 构建，并深度适配国产高性能车规芯片 —— **芯擎科技「龍鹰一号」系列平台**。

## 🚗 项目愿景

推动中国智能汽车基础软件生态发展，提供符合功能安全（如 ISO 26262）要求、支持混合关键性系统（Mixed-Criticality Systems）部署的开源车用 OS 基座，助力主机厂与 Tier1 快速构建下一代智能座舱与域控制器。

## 🔧 核心特性

- **微内核架构**：基于 L4Re + Fiasco.OC 微内核，实现最小可信计算基（TCB），提升系统安全性与可靠性。
- **硬件虚拟化支持**：利用 ARM TrustZone 与硬件辅助虚拟化技术，在龍鹰一号 SoC 上实现多 Guest OS（如 Linux、Android、RTOS）安全隔离共存。
- **车规级适配**：完整支持龍鹰一号平台（7nm 制程，8 核 A76/A55 CPU、14 核 GPU、8 TOPS NPU），包括中断控制器、内存管理、时钟与电源管理等关键驱动。
- **模块化设计**：内核、设备驱动、虚拟机监控器（VMM）、用户态服务均可按需裁剪，适用于从信息娱乐到高阶智驾的不同场景。
- **开源开放**：遵循宽松开源协议（建议 Apache 2.0 ），欢迎社区共建。

## 📦 组件概览

- `solid-fiasco`：L4Re/Fiasco.OC 微内核的車規增强版本  
- `solid-uvmm`：轻量级 Type-1 虚拟机监视器，支持 VirtIO 与直通设备  
- `solid-platform-sieagle`：龍鹰一号 SoC 板级支持包（BSP）  
- `solid-demos`：多 OS 共存演示（如 Android + AUTOSAR CP）

> **关于芯擎龍鹰一号**  
> 龍鹰一号是中国首款 7nm 车规级高算力智能座舱芯片，由芯擎科技（SiEngine）研发，集成 88 亿晶体管，支持 AEC-Q100 Grade 3 标准，已量产搭载于领克 08 等车型。

> **关于 L4Re**  
> L4Re 是一个开源的微内核操作系统框架，源自德国德累斯顿工业大学（TU Dresden），广泛用于嵌入式、安全关键与虚拟化场景，是 PikeOS、seL4 等商业/学术系统的同源技术路线。
